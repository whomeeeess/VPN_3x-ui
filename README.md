# **1. Правильный выбор VPS**

 Что такое VPS

 VPS (Virtual Private Server) — это удалённый сервер, на котором ты будешь поднимать свой VPN.

 **На что обращать внимание:**

**1. Локация сервера**
 ВАЖНО! местонахождение вашего сервера не должна быть в россии или китае (местах где действуют блокировки), иначе ничего работать не будет.
   Хорошие варианты:
    Германия
    Нидерланды

**2. Характеристики (минимум)**

CPU: 1 ядро (лучше 2)
RAM: 1–2 ГБ
Диск: 10–20 ГБ SSD
Скорость сети: от 100 Mbps


**3. ОС (операционная система)**
При создании VPS выбирай: Ubuntu 22.04 на нем вся и будет работа




# **2.Подключение к серверу**

**После покупки VPS тебе дадут:**

IP: ********

логин: root

пароль: *******

**чтобы зайти на сервер нам нужно**
 1. на своем пк нажимаем сочетание клавишь Win+R 
 2. вводим: **cmd**
 3. в открывшейся командной строке пишем:
  ```bash
  ssh root@ВАШIP
  ```



1. введите в командной строке комманды (данные комманды обновят все что нужно ): 
```bash  
apt-get update -y 
apt-get upgrade -y
``` 
2. установка необходимых пакетов, вводим:

```bash
apt-get install wget curl -y
```

3. Установка панели 3X-UI

для установки панели на сервер запустите выполнение скрипта командой:
```bash  
bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)
```
Во время установки скрипт задаст вопрос:

Would you like to customize the Panel Port settings? (If not, a random port will be applied) [y/n]:
Отвечаем "y" если хотите установить порт сами, или "n" тогда скрипт установит порт самостоятельно

Не устанавливайте стандартные порты 22, 80, 8080. Рекомендуется выбрать другой порт!


4. После выполнения этих шагов установка панели будет завершена скрипт сообщит Вам данные для подключения такого вида:

```shell
Username: 4grsfdea

Password: 3cewscer

Port: 3333

WebBasePath: dwaWJdnswlwmr

Access URL: http://192.168.0.0:3333/dwaWJdnswlwmr
```

После выполнения этих шагов установка панели будет завершена и можно подключиться через браузер к панели по пути|
``` shell
http://192.168.0.0:3333/dwaWJdnswlwmr #если у вас в Access URL протокол "https" просто уберите "s" и сможете зайти!
```
